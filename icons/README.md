# Icons

To generate the extension icons:

1. Open `generate-icons.html` in your browser (located in the parent directory)
2. Click "Download All Icons" button
3. Save the generated PNG files (icon16.png, icon48.png, icon128.png) to this directory

## Icon Design

The icon features:
- Red circular background
- White music note symbol
- White diagonal slash (blocking symbol)
- White border circle

This visually represents the extension's purpose: blocking/skipping music tracks.

## Required Sizes

- **16x16**: Shown in the Chrome toolbar (favicon size)
- **48x48**: Used in the Extensions management page
- **128x128**: Used in the Chrome Web Store and installation dialog
